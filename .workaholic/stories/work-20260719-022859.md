# Grow plggmatic: per-component theming slots + generalized horizontal runway

_Mission: grow-plggmatic-as-the-reference-framework — replanned and driven overnight by `/monitor` (run monitor-20260723-011758)._

## 1. Overview

The mission's central "express the reference generally, not special-cased" goal
had one primary gap: demo1 restyled 16 framework `pm-*` classes by name and owned
an 85-line DOM column-advance effect — both framework capabilities living in the
example. This closes both: theming moves to declared slots, and the
unbounded-depth horizontal runway becomes a framework capability. The mission was
also linked to its strategy (`plgg-horizontal-orientation-ui-stack`) with two new
acceptance criteria.

## 2. What changed

**Theming slots.** Added a `ComponentSlot` vocabulary + `slotSelector` (the
framework owns the `pm-*` strings), a `SlotStyle` + `asSlotStyle` caster (the
`asPalette` pattern), a `slotCss` emitter, and a `Theme.slots` field (default
`[]`). Migrated demo1's 30+ appearance overrides into `demo1/theme.ts` slot data
and removed them from `demo1/styles.ts`.

**Horizontal runway.** Added `runwayCss(theme)(opts)` (scroll-at-every-width
strip + `min-width:snap` `::after` flex spacer) and an `advanceColumns(theme)(done)`
Cmd (faithful relocation of demo1's DOM seek-head effect), plus row/col width
slots; reduced demo1's ~85-line `advanceColumnsCmd` to a one-line framework call.

## 3. Outcome

- **Zero `.pm-` selectors** remain in `demo1/styles.ts` (grep-verified) — the
  joint acceptance of both tickets.
- Visual identity proven by a rule-set reconstruction diff:
  `slotCss + runwayCss + demo1Css` emits the **identical 75-atomic-rule set** as
  the pre-refactor stylesheet (modulo an intentional, behaviour-preserving
  `--bo-*` → `--pm-runway-*` internal-var rename).
- The "depth does not consume the viewport" invariant is asserted structurally in
  `runwayCss.spec` (scroll container + trailing flex spacer); plgg-test's DOM has
  no layout engine for a true pixel measurement, noted transparently.
