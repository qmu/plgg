# Bump plggpress to 0.0.4 and publish for plggmatic docs

## Summary

Bumped `plggpress` from 0.0.3 to 0.0.4 and published it to npm so plggmatic's documentation site can consume the current source. The registry was stale at 0.0.3 while `packages/plggpress/src` had drifted; npm forbids republishing a version, so the current plggpress reached the registry as 0.0.4. Merging this PR brings main's source version into agreement with the published artifact.

## Key Changes

- `plggpress@0.0.4` published to npm (dist-tag `latest`), unblocking plggmatic's docs to depend on `^0.0.4`; its `file:` deps published as resolvable `^` ranges
- Version bump only — no plggpress source change on this branch
