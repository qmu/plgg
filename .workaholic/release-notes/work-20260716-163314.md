# Re-extract plgg-mcp as the content-free MCP protocol substrate, etc

## Summary

The MCP protocol substrate is its own package again: `plgg-mcp@0.0.2` carries the tool registry, dispatch, JSON-RPC framing, transports, and stdio seam with `node:sqlite` provably absent from its module graph, while `plgg-cms@0.0.3` keeps the content tools behind an unchanged barrel. The post-PoC integration roadmap also lands as a plan plus eight tickets.
