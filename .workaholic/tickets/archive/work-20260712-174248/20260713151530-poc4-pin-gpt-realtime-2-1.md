# PoC 4: pin the Realtime model to gpt-realtime-2.1

## Overview

Developer directive (2026-07-13, before live judging): PoC 4 inherited
PoC 3's `gpt-realtime` ALIAS, which the live model list shows still
resolving alongside the original GA snapshot (`gpt-realtime-2025-08-28`)
while `gpt-realtime-2.1` / `-2.1-mini` are available to the key. Pin
**gpt-realtime-2.1** explicitly — the newer snapshot's stronger
tool-calling discipline is exactly what `edit_file` (whole-document
rewrites) leans on. PoC 3 stays untouched (already judged and closed).

## Key Files
