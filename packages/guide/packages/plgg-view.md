# plgg-view

::: info Page stub

`plgg-view` is a minimal Elm Architecture (TEA) for the
browser: a typed `Html<Msg>` view tree (Elm-style
hyperscript builders, no JSX), pure `sandbox` /
`application` runtimes, and SSR `renderToString`. The
same `Model` / `update` / `view` program renders
server-side and client-side. Built on plgg only.
