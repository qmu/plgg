# 会社概要


## 当社について

当社は株式会社くむと申します。Web サービスや業務システムの受託開発・運用支援を行う東京の小さな会社です。

当社は、ソフトウェアを書いたり、システムを設計することに楽しさややりがいを感じる技術屋で、お客様の IT 戦略推進をプロとしてご支援します。しかし本音を申し上げれば、まだ「これは十分だ」と言い切れるような仕事を成し得たことはなく、その時最善と考えたご提案が後になって悔やまれることは多々あります。

しかしこれはお客様の事業自体が、様々な困難を解決し高度化されたものであり、その領域の専門家であるお客様を超えて良い成果を出すことは、そもそも挑戦的なことであると当社は自覚すべきと考えます。

またソフトウェア開発自体が本質的に困難なことである、と当社は考えます。ただ困難であるだけでなく、細分化・高度化が進み、変化のスピードも早く、新たな開発プロジェクトはいつも野心的な問題解決が求められます。

さらに今後 Claude Code などの AI エージェントを使い、お客様自身が問題解決を進められる近未来において、もう当社のソフトウェア技術職には「解決できる問題」がわかりやすく依頼されることなどなく、困難な課題に果敢に挑む他、プロとして生きる道はないものと考えます。

だからこそ当社は誠実に学び、地道に技術を磨き、お客様の事業理解に努め、AI による最新の開発アプローチを積極的に取り入れ、これからの時代においてもお客様に選ばれる実力を磨きます。
